# Financial Market Crash Prediction

## Overview
A machine learning project that predicts whether a stock market 
crash will occur based on historical economic indicators from 
USA (1978–2022).

## Dataset Features
- Unemployment Rate
- CPI (Consumer Price Index)
- P/E Ratio
- Stock Prices (Open, High, Low, Close)
- Industrial Production
- Treasury Rate
- Target: Crash (0 or 1)

## Tools & Technologies
- Python
- Pandas & NumPy
- Matplotlib & Seaborn
- Scikit-learn
- XGBoost

## Results
- Model: XGBoost Classifier
- Accuracy: 91.6%
